# Problem 3C: literal published predicate

The displayed predicate in the 2019 prize announcement literally negates the
existence of a finite exact machine whose runtime has finite `limsup t(n)/n`.
For nonnegative finite runtimes this excludes every exact `O(n)` algorithm,
not just sublinear algorithms.

This formulation is tracked because it is published, but it is not silently
substituted for the natural-language shortcut question. The source notation,
formal transcription, and citations are in `theory_literature_review.md`.

Status: `inconclusive`; intended contest interpretation requires external
clarification from the prize committee.
